# 🎨 Colour Analysis App

A modern web application that helps users discover their **perfect seasonal colour palette** based on their natural features.

🔗 **Live Demo:**  
https://mnejra03.github.io/ColourAnalysisApp/

---

## ✨ Features

- 🔐 User authentication (Sign up / Login)
- 🧠 Interactive colour analysis quiz
- 🎨 Personalized seasonal results (Spring, Summer, Autumn, Winter)
- 💄 Styling & makeup recommendations
- 💾 Save analyses to user profile
- 📱 Fully responsive design
- 🌍 Works across multiple devices

---

## 🛠️ Tech Stack

- **Frontend:** React (Vite)
- **Routing:** React Router
- **Backend:** Supabase (Auth + Database)
- **Styling:** CSS
- **Deployment:** GitHub Pages

---

## 🚀 How It Works

1. User creates an account or logs in  
2. Completes a short colour analysis quiz  
3. App determines their seasonal colour type  
4. Displays:
   - Season & subtype  
   - Colour palette  
   - Styling tips  
   - Celebrity matches  
5. User can save results to their profile
