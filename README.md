# Seasonal Colour Analysis App

AI-powered seasonal colour analysis. Upload a photo or take a quiz to discover your perfect palette.

## Stack
React + Vite · Supabase · Claude AI · Netlify

## Setup

1. `npm install`
2. Create `.env`:
```
VITE_SUPABASE_URL=...
VITE_SUPABASE_ANON_KEY=...
ANTHROPIC_API_KEY=...
```
3. Run `supabase-schema.sql` in Supabase SQL Editor
4. `npm run dev` → open http://localhost:5173

## Deploy
Push to GitHub → connect to Netlify → add env variables → done 🚀
